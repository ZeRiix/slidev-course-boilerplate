1. Create a `Student` type with a name and a grade.
2. Write an `isPassing` function that returns `true` when the grade is greater than or equal to 10.
3. Filter a list of students to keep only passing results.
4. Return an array of readable messages.

<TheHint title="Hint">
Start by typing a single value, then generalize with `Array.prototype.filter` and `Array.prototype.map`.
</TheHint>
